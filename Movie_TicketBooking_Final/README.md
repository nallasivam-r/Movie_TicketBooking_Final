\# Movie Ticket Booking Project

&nbsp;

A Java web application for booking movie tickets online.

&nbsp;

---

&nbsp;

\## Features

&nbsp;

\- Browse available movies and showtimes

\- Select seats for a specific show

\- Book tickets and generate a booking confirmation

\- View booking history

\- User-friendly interface with responsive design

&nbsp;

---

&nbsp;

\## Technologies Used

&nbsp;

\- Java

\- JSP / Servlet

\- MySQL Database

\- HTML, CSS, JavaScript

\- Apache Tomcat (Web Server)
